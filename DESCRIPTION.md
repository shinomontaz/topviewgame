# TopViewGame - Roguelike Dungeon Crawler

A classic turn-based roguelike where every move matters.

![image](screen.png?raw=true)

## 🎮 Gameplay

You wake up in a mysterious dungeon with no memory of how you got there. The rules are simple but unforgiving:

- **One action per turn**: Move OR Attack
- **Enemies follow the same rules**: They can only act when you act
- **Permadeath**: Every decision counts
- **Procedurally generated dungeons**: Each playthrough is unique

## 🎯 Goal

Find the exit and escape the dungeon alive. But beware - monsters lurk in every shadow, and resources are scarce.

## 🕹️ Controls

- **Arrow Keys / WASD**: Move
- **Mouse Click**: Move to location (pathfinding)
- **Space**: Wait/Skip turn
- **ESC**: Pause menu

## 💀 Features

- Turn-based tactical combat
- Field of view (FOV) system
- Procedural dungeon generation
- Multiple enemy types with unique behaviors
- Animated sprites

## 📦 Installation

1. Download the binary for your OS
2. Run the executable
3. Survive!
